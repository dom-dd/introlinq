Install the widget
Paste one line of code into your blog and the widget activates automatically on every article.
Your embed code
This is unique to your account — do not share it
1
Install the "Insert Headers and Footers" plugin
In your WordPress admin, go to Plugins → Add New and search for Insert Headers and Footers by WPBeginner. Install and activate it.

This is the safest way to add code — it doesn't require editing theme files and survives theme updates.
2
Open the plugin settings
Go to Settings → Insert Headers and Footers in your WordPress admin sidebar.
3
Paste your embed code in the Footer box
Find the Scripts in Footer section and paste your embed code there. Then click Save.
4
Done — visit any article to confirm
Open a blog post with at least a few paragraphs of text. After a few seconds, you should see phrases highlighted in your articles when there's a matching expert.
Note: The widget only activates on article/post pages with substantial content (150+ words). It won't appear on your homepage, category pages, or short posts.
1
Open Ghost Admin settings
Log in to your Ghost admin panel and click the Settings icon (gear icon) in the bottom-left sidebar.
2
Go to Code injection
In Settings, scroll down and click Code injection.
3
Paste in the Site Footer box
Paste your embed code into the Site Footer text area (not the header). Then click Save.
4
Done — open any post to verify
Visit a published post with a good amount of text. Phrases matching available experts will be subtly highlighted.
Ghost Pro users: Code injection is available on the Starter plan and above. If you're on a lower tier, you may need to upgrade or use a custom theme.
1
Open Project Settings
In the Webflow Designer, click the W logo in the top-left corner, then go to Project Settings.
2
Go to the Custom Code tab
Click the Custom Code tab at the top of Project Settings.
3
Paste in the Footer Code box
Scroll to the Footer Code section and paste your embed code there. Click Save Changes.
4
Publish your site
Click Publish in the Designer to push the changes live. The widget won't appear in the Webflow preview — you need to view the published site.
CMS collections: If your blog posts are in a Webflow CMS collection, the footer code injection above covers all of them automatically. No per-page setup needed.
1
Go to Settings
In your Squarespace admin, click Settings in the left sidebar.
2
Open Advanced → Code Injection
Scroll to the Advanced section and click Code Injection.
3
Paste in the Footer box
Paste your embed code into the Footer text area. Click Save.
4
Done — visit a blog post to verify
Open a published blog post. You should see the widget running on posts with enough content.
Squarespace plan: Code Injection is available on the Business plan and above. It's not available on the Personal plan.
1
Find your site's HTML template
Open the main layout or template file that wraps all your pages — usually called layout.html, base.html, _layout.liquid, or similar depending on your platform.
2
Paste just before </body>
Find the closing </body> tag and paste your embed code immediately before it. It should look like this:

... rest of your page ...
<script src="https://www.introlinq.com/widget.js" data-publisher="your-slug"></script>
</body>
3
Deploy and verify
Deploy your changes and open a content-rich page. The widget loads asynchronously so it won't slow down your site.
Using a tag manager? You can also paste the embed code into Google Tag Manager or similar. Add it as a Custom HTML tag, set it to fire on "All Pages", and deploy.
How to verify it's working
Open a published article on your site with at least 3–4 paragraphs of text.
Wait 2–3 seconds for the widget to run. If there's a matching expert, a phrase in the article will be subtly underlined.
Hover over a highlighted phrase — an expert card should appear.
Check your IntroLinq dashboard → the Matches tab should show a new entry within a minute.
No highlights? The article may not match any available experts yet — try a different article or check the Widget tab to adjust your match sensitivity.

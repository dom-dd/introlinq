IntroLinq WordPress Plugin
One-click install — no code editing needed. Enter your Publisher ID and you're done.
Download plugin
1
Download and install the plugin
Download the plugin above. In your WordPress admin, go to Plugins → Add New → Upload Plugin, select the downloaded introlinq.zip file, and click Install Now. Then click Activate.
2
Enter your Publisher ID
Go to Settings → IntroLinq in your WordPress admin sidebar. Enter your Publisher ID (shown in the embed code box above) and click Save Publisher ID.
3
Done — visit any article to confirm
Open a blog post with at least a few paragraphs of text. After a few seconds, you should see phrases highlighted when there's a matching expert.
Note: The widget only activates on article/post pages with substantial content (150+ words). It won't appear on your homepage, category pages, or short posts.

1
Open your Wix dashboard and go to Settings
Log in to your Wix account, select your site, and click Settings in the left sidebar.
2
Open Custom Code
Scroll down to the Advanced section and click Custom Code.
3
Add a new code snippet
Click + Add Custom Code. Paste your embed code in the text area. Set the name to "IntroLinq", the placement to Body — end, and apply it to All Pages. Then click Apply.
4
Publish your site
Click Publish in the Wix editor. The widget won't appear inside the Wix Editor preview — you need to view the live published site.

1
Open your GTM container
Go to tagmanager.google.com and open the container for your website.
2
Create a new tag
Click Tags → New. Give it the name "IntroLinq Widget".
3
Set Tag Configuration to Custom HTML
Click Tag Configuration, choose Custom HTML, and paste your embed code in the HTML box.
4
Set the trigger to All Pages
Click Triggering and select All Pages. Click Save.
5
Submit and publish the container
Click Submit in the top-right corner, add a version name like "Add IntroLinq", and click Publish. The widget will go live immediately across your whole site.
Already using GTM? This is the fastest path — no changes to your site's code needed, works across WordPress, Wix, Squarespace, or any platform that has GTM installed.
1
Find your site's HTML template
Open the main layout or template file that wraps all your pages — usually called layout.html, base.html, _layout.liquid, or similar depending on your platform.
2
Paste just before </body>
Find the closing </body> tag and paste your embed code immediately before it. It should look like this:

... rest of your page ...
<script src="https://www.introlinq.com/widget.js" data-publisher="your-slug"></script>
</body>
3
Deploy and verify
Deploy your changes and open a content-rich page. The widget loads asynchronously so it won't slow down your site.

How to verify it's working
Open a published article on your site with at least 3–4 paragraphs of text.
Wait 2–3 seconds for the widget to run. If there's a matching expert, a phrase in the article will be subtly underlined.
Hover over a highlighted phrase — an expert card should appear.
Check your IntroLinq dashboard → the Matches tab should show a new entry within a minute.
No highlights? The article may not match any available experts yet — try a different article or check the Widget tab to adjust your match sensitivity.
